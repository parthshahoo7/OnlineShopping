Instruction

Shop Products

If you want purchase a product you need to click on the Shop Products which is a subItem of the PRODUCTS. The site will then redirect you to a new page which will display all the products contained within our current inventory. Please insert the quantity of the product you wish to buy and then add it to your cart. Note: your order will not be placed until you checkout from your cart.

Add Product

If you would like to add a new product then you need to click on the Add Product which is a subItem of the PRODUCTS. You need to fill out the form with the product information that you want to add.Note: If you are a customer, you will not be able to add a product. This is reserved for administrators and employee accounts only.

Delete Product

If you would like to delete an existing product then you need to click on the Delete Product which is a subItem of the PRODUCTS You need to provide the Product Id to delete that product. Note: If you are a customer you will not be able to delete a product. This functionality is reserved for administrator only.

Search Product

If you would like to Search an existing product then you need click on the Search Product which is a subItem of the PRODUCTS. You will be asked to specify the ID of the product you would like to search.you can use this functionality from any page. It is already included in header. Note: If you are a Employee or Administrator accessing this site, you will not be able to search a Product. This functionality is reserved for Customers only.

Delete Customer

If you would like to delete an existing Customer Account then you need to clickDelete Customer. from the Menu. You will be asked to specify the Email ID of the customer you would like to delete. Note: If you are a customer or Employee accessing this site, you will not be able to delete a customer Account. This functionality is reserved for administrator accounts only.

View Customer Purchase Orders

If you would like to view the purchase orders, then click on VIEW PURCHASE ORDERS from the menu. If you haven't signed in the system will prompt you for your user name and password. Once you've signed in you will be able to view all customer purchase orders. Note: If you are a customer or employee accessing this site, you will not be able to see this feature. This functionality is reserved for administrator only.

Order History

If you would like to view the Order History, then click on Order History from the menu. If you haven't signed in the system will prompt you for your user name and password. Once you've signed in you will be able to view all your purchase orders. Note: If you are a Administrator or employee accessing this site, you will not be able to see this feature. This functionality is reserved for Customer only.

View Cart

If you would like to view your cart, then click on View Cart from the menu. If you haven't signed in the system will prompt you for your user name and password. Once you've signed in you will be able to view your cart. Note: If you are a Administrator or employee accessing this site, you will not be able to see this feature. This functionality is reserved for Customer only.
